Title: Student Analyzer frq project
By: Mousa Elkomy 11B2

Project Overview:

this analyzer class is designed to analyze collection of data of students from an arraylist
it can perform various tasks on arraylists of student type

explanations of algorithms:

Conditional counting- Loops through the lsit and increments the counter if a student mark exceeds a threshold
Safe removal- uses reverse for loop to remove students below a given mark. iterates from end to prevent skipping students
Maximum finder- finds the student with highest mark using a for loop
Duplicate detector- uses a nested loop to find if any students have same name
Pattern verification- checks if the list follows a strictly increasing order

Challenges:
Static vs nonstatic when calling the methods in main method
reverse loop algorithm where there was a risk of skipping iterations of few students
